# Vue 3.2 + Typescript + Vite Template

This template is a minimal achieve use typescript + vue3.2 + vite + pinia.Check out the original repository [Vitesse](https://github.com/antfu/vitesse) to learn more.

## Features

- Vue3
- Vite
- Vue-Router
- Pinia
